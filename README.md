## Requirements
* python >= 3.6
* tkinter >= 8.6
* matplotlib >= 3.2.1

## Installation
1. Create a Virtual Environment
2. Use package manager to install setup.py file
```bash
pip3 install .
```

## Run Test
1. Existing unit tests are available in test.py file
2. Run test.py
```bash
python3 -m unittest test
```
3. Add your tests (if you like)
